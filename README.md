# 康姿健屯門美容

屯門美容一人小店官方網站，內容來自 Instagram [@hong_chi_kin](https://www.instagram.com/hong_chi_kin/)。

## 頁面

- `index.html` — 首頁
- `about.html` — 關於我們
- `services.html` — 療程服務
- `gallery.html` — 作品分享
- `contact.html` — 聯絡我們

## 本地預覽

在專案目錄執行：

```bash
python3 -m http.server 8080
```

然後開啟 http://localhost:8080
