# 康姿健屯門美容

屯門美容一人小店官方網站，內容來自 Instagram [@hong_chi_kin](https://www.instagram.com/hong_chi_kin/)。

## 頁面

- `index.html` — 首頁
- `about.html` — 關於我們
- `services.html` — 療程服務
- `blog.html` — 2026 護膚知識博客
- `gallery.html` — 作品分享
- `contact.html` — 聯絡我們

## 建立博客

從 Google 試算表更新內容後執行：

```bash
node js/build-blog.js
```

## 本地預覽

```bash
python3 -m http.server 8080
```

## SEO

- `sitemap.xml` — 網站地圖
- `robots.txt` — 搜尋引擎指引
- 每頁含 JSON-LD（BeautySalon / BlogPosting）及 geo 標籤
